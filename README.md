# Image-processing-using-numpy
          The program is menu-based with the following top-level features:

              1. Load Image
              2. Edit Image
              3. Quit ‘Edit Image’ is able to do the following using numpy operations:
              4. Horizontal Flip
              5. Vertical Flip
              6. Invert Colors
              7. Convert to Grayscale
              8. Quit Editing
